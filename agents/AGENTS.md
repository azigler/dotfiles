## Meeting Transcripts (`meet` CLI)

A CLI tool for searching and reading meeting transcripts is available at `~/.local/bin/meet`.

Quick usage:
- `meet summary` -- overview of available transcripts
- `meet list --titles --last 5` -- recent meetings
- `meet search "topic" -i` -- find discussions about a topic
- `meet read --latest --body` -- read the latest transcript

Use the `/meet` slash command for full reference, or run `meet --help`.

Setup: `bash /home/ubuntu/webhook-macwhisper/scripts/agent-setup.sh`
